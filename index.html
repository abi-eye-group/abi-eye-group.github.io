Controller Options

The control panel has the following options:



Main
Stimulus
[ default: disks ] Stimulus choices e.g. "disks" | "bars" | "sinusoids"
logMAR
[ default: 1.0 logMAR ] logMAR specification is: 1) disks: angular size of central disk, 2) bars: stripe width, 3) sinusoids: wavelength/2
Speed
[ default: 5.0 deg/sec ] Horizontal drift speed for stimulus
Direction
[ default: left ] Horizontal drift direction.
Distance
[ default: 100 cm ] Distance of pbserver from display
Sweep
[ default: true ] sweeping enabled. {[true]|false}


Sweep
Step Duration
[ default: 1.0 sec ] Time spent per logMAR level
Step Size
[ default: 0.1 logMAR ] Step Size


Stimulus Specific Options


Disks
Ratio
[ default: 1:2 ] Disk center:surround diameter ratio.
Central Intensity
[ default: 0.75 ] Central disk intensity (normalized pixel intensity)
Surround Intensity
[ default: 0.45 ] Surround disk intensity (normalized pixel intensity)
Spacing
[ default: 2 deg ] Angular distance between disks


Bars/Sinusoids
Contrast
[ default: 1 ] Michelson contrast


Display Options
Height (cm)
[ default: cm ] Physical screen height
Width (cm)
[ default: cm ] Physical screen width
Height (px)
[ default: screen.width px ] Screen height in logical pixels
Width (px)
[ default: screen.width px ] Screen width in logcial pixels
Device Pixel Ratio
[ default: 1 ] Conversion from logical to physical pixels
